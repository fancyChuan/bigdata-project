## 用户画像系统

用户画像建模就是对用户“打标签”，有三种类型；
- 统计类：性别、年龄、近7天活跃时长、活跃天数等
- 规则类：比如“消费活跃”用户的口径定义为“近30天交易次数>2”（运营人员和数据人员共同协商确定）
- 机器学习数据挖掘类：比如购买商品偏好、用户流失意向等（成本高，一般这类标签占比较少）
